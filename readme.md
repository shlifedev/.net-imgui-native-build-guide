
This is cimgui build guide for ImGuI.Net

# 1. Install Cmake & Git
 https://cmake.org/  
 https://git-scm.com/ 
  
 # 2. Build
 ```
 git clone -b additional_libraries --single-branch --recursive https://github.com/TillAlex/ImGui.NET-nativebuild
 cd ImGui.NET-nativebuild
 ci-build.cmd
 ```
